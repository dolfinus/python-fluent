python-fluent¶

The python-fluent project contains several packages to bring Project Fluent to Python. Visit the Github project for the full list. Three packages in particular are of general interest and documented here.

fluent.syntax¶

fluent.syntax is the package to use for tooling, analysis, and processing of Fluent files.

fluent.runtime¶

fluent.runtime is the reference runtime implementation for Fluent in Python.

fluent.pygments¶

fluent.pygments can be used to add syntax highlighting for Fluent files to Sphinx documentation.
